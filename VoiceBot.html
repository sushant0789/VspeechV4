How VoiceBot Works

  1. Caller asks a questions
  2. Questions audio streamed for conversion
  3. Question returned as text
  4. Text sent for NLP/NLU Analysis
  5. Response Corrected Text Return
  6. Text sent to be synthesized to TTS
  7. Synthesized Audio response returned
  8. Audio response streamed to caller
